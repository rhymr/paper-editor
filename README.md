# Rhymr

Rhymr is a modern, AI-powered lyric and poetry writing editor with rhyme and syllable tools, chat, and file management. Write, edit, and export your creative work easily.

## Usage

See [`https://rhymr.github.io/`](https://rhymr.github.io/editor.html)
